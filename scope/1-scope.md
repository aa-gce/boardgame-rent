# Project Scope Statement: Board Game Rental System
The Online Board Game Rental System is a web-based platform that connects board game owners(Lenders) with renters(Borrowers). The system will include:
*   **User Management:** Registration, login, profile management, and a trust/rating system for users.
*   **Game Catalog & Listing:** Game owners can upload photos, descriptions, physical conditions, rental rates, deposit values, and location. Renters can search, filter, and view games.
*   **Order & Rental Workflow:** Booking requests, approval/rejection by owners, delivery tracking status, and return confirmation.
*   **Payment & Escrow System:** Integration of a secure gateway to handle rental fees, hold deposits (escrow), and process refunds or fine deductions.
*   **Admin Dashboard:** Tools for administrators to manage users, resolve disputes, monitor transactions, and handle reports of damage or late returns.

### ✅ Project Scope (The work required to deliver the product)
*   Requirement gathering and system analysis.
*   UI/UX design (wireframes and mockups).
*   Database design and backend API development.
*   Frontend development (responsive web or mobile application).
*   Integration of payment gateway and notification system.
*   Testing (unit testing, system testing, and user acceptance testing).
*   Deployment on a cloud server.
*   Project management activities (planning, scheduling, and documentation).

### ❌ Out of Scope (What is NOT included in this project)
*   **Logistics & Physical Delivery:** The system will not provide its own delivery fleet or couriers. Users must arrange delivery (in-person meetups or using third-party post/couriers at their own cost).
*   **Game Maintenance/Repair:** The platform will not physically repair or clean damaged games; it only facilitates the dispute/fine process.
*   **Buying/Selling Games:** The platform is strictly for renting; direct sales of board games will not be supported in this phase.
